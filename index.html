Projects

Projects

Below are the sample Data Analytics projects on SQL, Python, Power BI & ML.

Lok Sabha Elections 2019 - Data Analysis using Python, SQL

Leveraged Pandas, Numpy, Matplotlib, Seaborn, and sqlite3 for comprehensive 2019 elections analysis, incorporating ETL, data collection, preprocessing, and exploratory data analysis. Applied advanced techniques including joins and window functions. Evaluated state-wise political party performance and election factors, while assessing opposition performance per state to identify areas for future electoral improvement.

Danny's Dinner Restaurant - Data Analysis using SQL

The SQL project analyzes customer transactions at Danny's Diner. Concepts like joins, aggregations, window functions, and conditional logic are employed. It calculates total spending, visit frequency, popular items, and member benefits. Techniques include point system calculations, membership status determination, and ordering analysis. The project provides insights into customer behavior and loyalty program effectiveness.

Sachin Tendulkar Power BI Dashboard

Developed a lively Power BI dashboard employing Power Query for ETL Operations. Explored Sachin's performance against various opponents with yearly breakdowns displayed in charts. Enhanced user experience with customized tooltips, field parameters, and slicers for smoother interaction. Additionally, utilized Data Analysis Expressions (DAX) for advanced calculations and data modeling, incorporating functions like COALESCE, BLANK(), ALL EXCEPT, and IFS for refined data processing.


Amazon Sales Power BI Dashboard

The Advanced Amazon Sales Dashboard in Power BI provides comprehensive visualizations of revenue, product sales, and customer behavior, leveraging advanced analytics and DAX functions. Customizable filters and real-time data integration enhance exploration. Sophisticated data modeling is pivotal in organizing insights, optimizing Amazon sales strategy and revenue, offering e-commerce professionals a powerful tool for informed decision-making and strategic growth.

Spotify Music Power BI Dashboard

The Spotify Music Analysis Dashboard leverages DAX (Data Analysis Expressions) functions within Power BI, enabling dynamic calculations and aggregations of crucial metrics like track popularity and artist trends. This integration empowers the dashboard to offer in-depth insights into music trends and user preferences through a user-friendly interface and diverse visualizations. Creating this dashboard not only provided hands-on experience but also honed proficiency in implementing new features within Power BI.

Electoral Bonds Power BI Dashboard

Data extraction from the Election Commission of India's (ECI) PDFs is facilitated by Python libraries like pandas and tabula-py. Extracted data undergoes cleaning in PowerBI using Power Query. The merge function is then employed to combine tables, enhancing data analysis. Key features include identifying top recipients, donor origins, and tracking donation trends over time, fostering transparency and informed decision-making.
